Aide sur l'utilisation du lecteur Cyberlibris

Navigation

Naviguez dans l'ouvrage grâce à :
  • La barre de défilement vertical (appelé aussi ascenseur)
  • La "Table des matières" accessible depuis le menu
    (pour sortir cliquez sur )

    • Zoomez ou dézoomez grâce aux boutons et

Impression

Cliquez sur puis cliquez sur "imprimer", vous aurez 2 possibilités :
  • Impression de la page courante (page sur laquelle vous êtes) cliquez sur
  • Impression de plusieurs pages successives : cliquez sur puis notez les pages à imprimer et cliquez sur

Pour ne plus voir cette fonctionnalité, cliquez sur .

Note

Cliquez sur puis sur :
  • Créez une note :
    • Cliquez sur , une note apparait
    • Complétez-la et sauvegardez-la en cliquant sur
  • Consultez votre note :
    • Cliquez sur , la note apparait en entier
    • Cliquez sur , vous ne verrez que le titre et le début de la note
    • Complétez votre note :
    • Cliquez sur le bouton de la note souhaitée, effectuez votre modification et
  • Supprimer une note:
    • Cliquez sur le bouton de la note correspondante, confirmez la suppression en cliquant sur

Pour ne plus voir la liste des notes, cliquez à nouveau sur .

Marque page

  • Créez un marque page : Cliquez sur pour marquer une page que vous souhaitez retrouver plus tard (elle devient verte).
  • Retrouvez une note : Cliquez sur , puis sur : toutes les pages marquées apparaissent, cliquez sur celle qui vous intéresse
  • Supprimez une note : Cliquez sur (le signet redevient gris), la page n'est plus marquée

Pour ne plus voir la liste de vos marques pages, cliquez à nouveau sur .

Recherche

Rechechez dans l'ouvrage un mot ou une expression :

Cliquez sur puis :

  • Notez l'objet de votre recherche dans le rectangle prévu à cet effet puis cliquez sur pour obtenir le résultat de vos recherches.
    NB : Les règles de recherche sont sensiblement les mêmes que sur Internet. Dans le cas d’une expression, il est nécessaire de la noter entre guillemets. Il est recommandé de penser à noter les accents.
  • Cliquez sur la page qui vous intéresse pour accéder à son contenu

Pour ne plus voir la liste de vos marques pages, cliquez à nouveau sur .
